# @mw-mono/header

A reusable React header component library.

## Installation

This library is part of the mw-mono monorepo.

## Usage

```tsx
import { Header } from "@mw-mono/header";

function App() {
  return <Header title='My Application' />;
}
```

## Props

- `title` (string, optional): The title to display in the header. Default: "My App"
- `className` (string, optional): Additional CSS classes to apply to the header

## Features

- Navigation links (Home, Products, Cart, Checkout)
- Customizable title
- Tailwind CSS styling
- TypeScript support
